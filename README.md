# 🚀 My Portfolio Website

## 🌐 Live Demo

## 📋 About
This is my personal portfolio website showcasing my skills, projects, and experience as a CS student at COMSATS University.

## ✨ Features
- Anti-gravity floating design
- Responsive layout
- Contact form with Supabase
- Modern UI with Tailwind CSS

## 🛠️ Tech Stack
- **Frontend:** React + Vite
- **Styling:** Tailwind CSS
- **Database:** Supabase
- **Deployment:** Vercel
